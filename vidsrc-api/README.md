# VidSrc API - Deploy to Vercel

1. Go to https://github.com/new and create a new repository
2. Upload the contents of this `vidsrc-api/` folder to that repo
3. Go to https://vercel.com/new and import your repository
4. Use default settings and click Deploy
5. After deployment, copy your Vercel URL (e.g. `https://your-app.vercel.app`)
6. Update `api/proxy/vidsrc.php` line ~17 with your Vercel URL
